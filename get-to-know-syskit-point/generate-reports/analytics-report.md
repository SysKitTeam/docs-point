# Analytics Reports

Reports in this article are useful for tracking user interaction with Microsoft 365 services, comparing adoption between departments, detecting which content and services are not being used. For some of these reports you will need an additional Analytics and Adoption modu.e

## Site Analytics Report

Site Analytics report gives you an overview of all your sites showing the total hits each site had in the last 7 days, the highest number of daily unique visitors and more.

### Generate File and Page Activities Report

- From SysKit Point home screen, click on the Reports tile.
- In the upper left corner click on the filter and select Analytics filter.
- Click on the Site Analytics Report


### Analytics Report Data 

Now that your report was successfully generated you can see the following data by the columns:

- Site Name - shows you the name of the site in the column, when selecting the row you can see the charts for Site Hits, Site Visitors and Site Storage.
- URL - gives you a direct link to the said site on your Microsoft 365 tenant.
- Total Hits - Shows you the total number of hits said site had in the last 7 days.
- Visitors - Shows you a total number of unique visitors to the site.
- Size - shows you the total size of the site
- Last Content Modified - shows you the last date something was modified on said site.

you can also choose to see the analytics for last 14, 30 days or you can choose a custom date period.

## Site Structure Report

Site Structure report gives you the ability to check the hierarchical structure for your sites down to the file level. Use this report when you want to display the structure of your Sharepoint environment. You can also Export the report and you can delete unique or stop inheriting permissions.

### Generate Site Structure Report

- From SysKit Point home screen, click on the Reports tile.
- In the upper left corner click on the filter and select Analytics filter.
- Click on the Site Structure report
- Now that you are redirected to Selection screen, choose a site for which you want to generate a report. You can choose to show content down to; Subsites, Document libraries & lists, Folders and Files and folders by selecting the dropdown in the additional filter on the bottom. 
- Click on the Run Report button.

### Site Structure Report Data

Now that your report was successfully generated you can see the following data by the columns:

- Name - shows you the name of the site and all the content in the structure down to files & folders.
- URL - clicking on it you are redirected to the exact location of the site or subsequent folders, subsites etc.
- Last Modified - shows the exact date something was modified on the site.
- Created On - shows the exact date when the site or some of its structure was created.

Site Structure report can also be exported to PDF and XLSX format and you can schedule it.

## Inactive Content Report

Inactive Content report gives you the ability to easily detect unused sites, Microsoft 365 Groups and Microsoft Teams in your Microsoft 365 environment. Use this report to always be aware of unused resources in your environment, so you can keep it clean and healthy.

### Generate Inactive Content Report

- From SysKit Point home screen, click on the Reports tile.
- In the upper left corner click on the filter and select Analytics filter.
- Click on the Inactive Content report.
- On the Options screen select for which type of resource you want to generate the report (Site, Microsoft 365 Group or Microsoft team) and click Run Report button.

### Inactive Content Report Data

Now that the report was sucessfully generated, you can see all the resources in your environment that is currently inactive. You can Export the report data into PDF and XLSX formats. 

If you select one or multiple resources you can also perform Lifecycle Management actions on thos resources (choose to Keep, Archive or Delete the resours) or you can Ask Owners to Renew said resources. When either of those actions are completed, said resource won't be visible on Inactive Content Report anymore.

When looking at the generated reports you can see the following columns:

- Name - shows you the name of the inactive resource
- Sharepoint, Exchange, Teams and Yammer activity column shows inactivity based on the last date some activity was made on said resource.
- Lifecycle Management - shows if the site was monitored for activity by SysKit Point Lifecycle Management.

## SharePoint Activity Report

SharePoint Activity report gives you a detailed overview of SharePoint usage. Report on the most active users, most popular sites, documents or pages. Use this report to track adoption of the SharePoint in your environment.

To see this report you need to have Adoption and Analytics module installed in SysKit Point.

### Generate SharePoint Activity Report

- From SysKit Point home screen, click on the Reports tile.
- In the upper left corner click on the filter and select Analytics filter.
- Click on the SharePoint Activity report.
- On the Selection screen, choose for which resources you want to generate a report, Select a time period on the bottom filter and click on Run Report button.

### SharePoint Activity Report Data

Now that the report is sucessfully generated you can see detailed graphs showing the activity of Users, Sites, Pages and Documents. See the Most Active Users, Sites, Top 5 pages, Top 5 Document visitors and Top 5 Documents visited. You can also see the number of Unique visitors, Most popular Days, which type of devices are visitor using etc.

## Most Popular Content Report

Most Popular Content report makes it easy to identify the most popular documents and pages in your Microsoft 365 environment.

To see this report you need to have Adoption and Analytics module installed in SysKit Point.

### Generate Most Popular Content Report

- From SysKit Point home screen, click on the Reports tile.
- In the upper left corner click on the filter and select Analytics filter.
- Click on the Most Popular Content report.
- On Selection screen choose a resource you want to generate report for, in the bottom filter select the time frame and choose if you want to see Pages or Documents, Click on the Run Report button.

### Most Popular Content Report Data

Now that the report is successfully generated you can see the most popular pages or documents in your microsoft 365 environment. You can also export the report into PDF and XLSX format or schedule the report. Columns that are seen by default are:

- Name - Shows you the name of the page or the document, when clicked on it you are redirected to the details page of that page or document in SysKit Point.
- Site Name - Shows the name of the resource where the most popular document is, when clicked on it you are redirected to site details page in Syskit Point.
- Hit Count - Shows the unique number of hits on the page or document.
- Unique Visitors count - Shows the number of unique visitors that have visited said page or document.

## Meeting Interactions Report

Meeting Interactions report gives you the ability to explore how people are interacting with each others in Microsoft Teams via 1:1 calls and meetings. Use the report to see how are users adopting Microsoft Teams in your environment.

To see this report you need to have Adoption and Analytics module installed in SysKit Point.

### Generate Meeting Interactions Report

- From SysKit Point home screen, click on the Reports tile.
- In the upper left corner click on the filter and select Analytics filter.
- Click on the Meeting Interactions report.
- On Selection screen choose for which company department you wish to generate a report and at the bottom filter select the time period, Click on the Run Report button.

### Meeting Interactions Report Data

Now that the report is generated you get to see the connection tree of all users in selected environment and how often or do they even interact with each others via 1:1 calls or meetings. You can also see the Activity tab where you see the most active users in that department.

## Network Interaction Details Report

Network Interaction Details reports gives you the ability to view details about members interactions via meetings or conversations.

To see this report you need to have Adoption and Analytics module installed in SysKit Point.

### Generate Network Interaction Details Report

- From SysKit Point home screen, click on the Reports tile.
- In the upper left corner click on the filter and select Analytics filter.
- Click on the Network Interaction Details report.
- On Selection screen choose for which company department you wish to generate a report and at the bottom filter select the time period, Click on the Run Report button.

### Network Interaction Details Report Data

Now that your report is generated you can view the details about members interactions be it via meetings or conversations. You can Export the report to the PDF or XLSX format or Schedule it. 

Network Interaction Details report is a tree type style report that shows the User as a Parent branch and all other users they interacted with as a child branch. 

Report shows departments for each user so you can see which departments are interacting with each other, it also shows how many Calls and Chat Messages were exchanged between said users.
