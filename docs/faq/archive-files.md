# Archive File

## Can end-users still find archived files?

Yes. Archived files remain visible in the SharePoint document library and are marked with an archived indicator. They can still be discovered through Microsoft 365 native search. However, **archived files mmust be reactivated before they can be opened or edited.**

## Can archives files be restored?

Yes. Archived files can be reactivated when access is needed. The time restoration takes depends on how recently the file was archived.
* Files restored within the first 7 days of being archived, are restored instantly.
* Files restored after the first 7 days of being archived, restoration can take up to 24 hours. 

Both end-users and admins can restore archived files. 

## Who can restore archived files?

There are two options for restoring files:

* End-users - users can restore files using Microsoft 365's native restore capabilities in SharePoint
* Administrators - admins can restore files from the site.level view in Syskit Point, which is useful for bulk restores
